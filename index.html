    emir - enhanced cross-platform display manager

    eMir is a display manager that provides efficient support for graphics coprocessors. eMir was forked from http://launchpad.net/mir as the '-ng' version of Canonical's Mir.

    For the rationale behind the fork, and why a Mir-based solution is superior to existing technology (e.g. X11, Wayland) please see the Announcing-eMir page on the wiki.
